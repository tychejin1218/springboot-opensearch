## [Spring Boot] Spring Boot와 OpenSearch Client 연동
- #### 개요
  - ##### Spring Boot와 OpenSearch Client를 연동하고, OpenSearch 서버와 통신하여 데이터를 저장, 삭제, 조회하는 등의 개발 환경을 구축

- #### 개발 환경
  - ##### IDE(IntelliJ IDEA 2023.3.4)
  - ##### JDK 17
  - ##### Spring Boot 3.3.1
  - ##### Gradle 8.8
